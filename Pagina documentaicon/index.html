    Para almacenanr un valor en la variable creada utilizamos el operador de asignacion "=".

    Escritura camel Case
  • Para declarar variables se utiliza la escritura "camel Case".

    • En camel Case la primera palabra va en miniscula y la primera letra de las siguientes palabras en mayuscula.

    var numeroTelefonico=15154872; var gustoDeHeladoFavorito=Frutilla; var estaturaDelIndividuo=1.75m;

    esta forma de escritura es la mas utilizada como metodo predeterminado de trabajar en JavaScript
